# Interface d’une application de jardinage – Projet IHM

## Description

Ce projet a été réalisé dans le cadre de l'enseignement d’Interaction Homme-Machine (IHM). Il consiste en la conception de l’interface d’une application de jardinage permettant aux utilisateurs de suivre leurs plantes, consulter des informations botaniques et planifier leurs activités de jardinage.

Le projet a été développé selon une approche centrée utilisateur, incluant l’analyse des besoins, la conception de prototypes et l’application des principes d’ergonomie et d’utilisabilité.

---

## Objectifs

- Concevoir une interface intuitive et facile à utiliser
- Appliquer les principes de l’Interaction Homme-Machine
- Réaliser un prototype d’application
- Concevoir une expérience utilisateur claire et ergonomique

---

## Fonctionnalités principales

- Carnet botanique pour le suivi des plantes
- Consultation d’une base de données de plantes
- Planification des tâches de jardinage
- Interface utilisateur interactive et intuitive
- Prototype UX/UI complet

---

## Technologies et outils utilisés

- Conception UX/UI
- Prototypage d’interface
- Design centré utilisateur
- Méthodes d’Interaction Homme-Machine
